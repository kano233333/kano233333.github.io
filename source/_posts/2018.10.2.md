# 2018.10.2

## 在动画时间内click

- 现象：

  ```js
  $('aaa').click(function(){
  	$('xxx').animate({
      	'xxx':'xxx';
  	},3000);  
      ...
  });
  ```

  设置了3000的动画时间，在3000动画时间里click  ===>  会出问题

- 解决

  ```js
  var animateEnd = false;
  var animateTime = 3000;
  $('aaa').click(function(){
      animateEnd = true;
  	$('xxx').animate({
      	'xxx':'xxx';
  	},animateTime);  
      ...
      if(animateEnd){
          setTimeout(function(){  //延时click
              animateEnd = false;
          },animateTime);
      }
  });
  ```
