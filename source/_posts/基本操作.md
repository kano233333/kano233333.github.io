## 查看数据库

```mysql
show databases;
```

## 新建数据库

```````mysql
create database dbname;
```````

## 删除数据库

`````mysql
drop database dbname;
`````

## 选择数据库

> 选择要操作的数据库，使用该命令后后面的mysql命令全部针对该数据库

````mysql
use dbname
````

## 查看表

````mysql
show tables;
````

## 查看表里的内容

```````mysql
show columns from tablename;
```````

## 创建表

```````mysql
create table tablename (...);
```````

## 删除数据库

````mysql
drop table tablename;
````

## 插入数据

```````mysql
insert into tablename (...) values (...);
```````
